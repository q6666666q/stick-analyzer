# 🎮 摇杆射击行为分析工具

> 一套用于事后分析 FPS 游戏（如 Apex Legends）射击行为的 Python 工具。 通过 XInput 记录手柄数据 → 自动分析射击稳定度 → 生成 AI 调参提示词。

## ✨ 主要功能

- **自动事件检测** — 无需外部时间戳，自动识别每次开火爆发
- **多维度分析** — 开火前稳定度、开火中稳定度、过冲次数、推杆量分布
- **可视化报告** — 每次开火生成波形图 + 总览统计图 + 文字报告
- **AI 调参提示词** — 自动生成包含分析数据的提示词，丢给 AI 即可获得针对性曲线建议
- **支持各种手柄** — 自适应不同曲线点数（2/4/6/8 点）、不同 RC 数值范围（±10 到 ±500）
- **GUI 友好** — 图形化界面，键位下拉选择，无需懂代码

## 📥 快速开始

### 方法 1：下载 EXE（推荐，零门槛）

1. 打开dist文件夹
2. 下载最新版的 `StickAnalyzer.exe`
3. 双击运行即可，不需要安装 Python

### 方法 2：源码运行

需要 Python 3.10+ 环境：

```bash

git clone https://github.com/你的用户名/stick-analyzer.git
cd stick-analyzer

# 安装依赖
pip install -r requirements.txt

# 运行

```

```bash
pip install pyinstaller
python build_exe.py
```

打包后的 `StickAnalyzer.exe` 在 `dist/` 目录下。

## 🚀 使用流程

### 第一步：录制摇杆数据

1. 打开软件，进入「① 录制摇杆数据」标签页
2. **选择你的开火键和开镜键**（下拉框选择，支持 16 种按键）
3. 填写元数据（RC 数值、动感强度等级、武器、场景）
   - 没有 RC 功能的手柄填 0 + 选「无 RC 功能」即可
   - 不同手柄 RC 范围不同（±10、±100、±500），按你 APP 里的实际比例选强度
4. 点击「● 开始录制」，正常打游戏
5. 完成后点击「■ 停止录制」

### 第二步：分析数据

1. 进入「② 分析数据」标签页
2. CSV 路径会自动填入（或手动选历史 CSV）
3. 点击「▶ 开始分析」
4. 查看实时分析报告

输出文件（与 CSV 同目录）：

- `*_report.txt` — 文字报告（含调参建议）
- `*_event_NN.png` — 每次开火的波形图
- `*_summary.png` — 总览统计图

### 第三步：生成 AI 调参提示词

1. 进入「③ 生成 AI 调参提示词」标签页
2. 提示词模板已自动嵌入分析报告
3. 在文本框中填入：
   - 你的手柄型号、曲线编辑方式、可调节点数
   - 你当前的曲线坐标
   - 你的体感痛点
4. 点「📋 复制全部到剪贴板」
5. 粘贴给 AI（推荐 Claude / ChatGPT）即可获得针对性的曲线调整方案

## 📊 分析指标说明

| 指标         | 含义             | 优秀  | 良好  | 一般  | 差    |
| ------------ | ---------------- | ----- | ----- | ----- | ----- |
| 开火前稳定度 | 瞄准时准星稳不稳 | <0.04 | <0.06 | <0.10 | >0.10 |
| 开火中稳定度 | 压枪过程稳不稳   | <0.04 | <0.06 | <0.08 | >0.08 |
| 反转次数     | 过冲严重程度     | <8    | <12   | <20   | >20   |

> 🔧 阈值会根据你的 RC 动感强度自动调整。

## 🔧 常见问题

### `Fire=0%` 即使按键也没反应？

检查录制时的开火键设置是否对。如果用的是扳机当开火，选「右扳机 (RT)」。

### 生成的图中文乱码？

确保系统装了 Microsoft YaHei 字体（Windows 自带）。

### EXE 被杀软误报？

PyInstaller 打包的 EXE 经常被误判，加入白名单或自己用源码打包即可。

### CSV 文件太大？

1 小时记录约 200-300MB，长时间游戏建议分段记录。

## 📝 关于设计原则

本工具基于这些核心原则设计：

1. **数据辅助，体感为主** — 数据用来发现具体问题，最终判断标准始终是体感
2. **不假设硬件** — 不同手柄的曲线点数、RC 范围、按键映射都不同，工具自适应
3. **不引导过度调参** — 频繁切换曲线会污染肌肉记忆，工具会提醒用户
